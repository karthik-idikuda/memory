# NEURON-X Omega

Self-sovereign permanent AI memory system with biologically-inspired architecture, custom SOMA-DB storage engine, Ebbinghaus decay curves for memory retention, and multi-layer contradiction detection.

## Features

- Custom SOMA-DB storage engine for persistent memory
- Ebbinghaus forgetting curve implementation for natural memory decay
- Contradiction detection across memory layers
- Memory injection and extraction pipelines
- Neural pathway-inspired memory linking
- REST API for external integrations
- Web-based memory exploration interface

## Tech Stack

Python, FastAPI, Anthropic Claude API, Custom DB Engine, React

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/memory.git
cd memory
pip install -r requirements.txt
```

### Usage

```bash
python main.py
```
